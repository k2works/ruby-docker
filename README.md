Ruby実行検証環境
===================

# 目的 #
コンテナを使っていつでも手軽にRubyの実行環境を検証できるようにする

# 前提 #
| ソフトウェア   | バージョン   | 備考        |
|:---------------|:-------------|:------------|
| ruby           |2.2.1    |             |
| docker         |1.9.0    |             |

# 構成 #
## 開発環境セットアップ

    docker build -t ruby-env .
    docker run -it -v $(pwd):/root/home --name work ruby-env
